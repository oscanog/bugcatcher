# User Guide

## How A User Starts

1. Link your Discord account from BugCatcher.
2. Go to an approved Discord intake channel.
3. Send your request with at least one image.
4. Follow OpenClaw's questions for organization, project, and verification details.

## What OpenClaw Will Ask

- which organization the request belongs to
- which project should receive the checklist batch
- clarifying QA questions if the image or request is ambiguous
- how to handle duplicates if matching checklist items already exist

## Duplicate Review

If OpenClaw finds possible duplicates in the selected project, she will summarize them and ask whether to:

- skip duplicates and submit only new items
- add all duplicates again
- review duplicate items one by one

## Result

After submission, OpenClaw returns:

- the created batch id
- the project name
- how many items were created
- how many duplicates were skipped or re-added
- a direct BugCatcher checklist link

## Important Rules

- image required
- project required
- only your real BugCatcher organizations and projects are allowed
- checklist batches created from Discord are attributed to your BugCatcher account
